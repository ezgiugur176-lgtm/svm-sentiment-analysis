# Analyse de Sentiment avec SVM

Ce dépôt contient l’implémentation en Python utilisée dans le cadre de ce mémoire, ainsi que le jeu de données intitulé *product data*.

Méthodologie :
- Vectorisation des commentaires textuels par la méthode TF-IDF
- Classification binaire à l’aide des Machines à Vecteurs de Support (SVM) avec noyau linéaire

Bibliothèques utilisées :
- pandas
- scikit-learn

Les données et le code sont fournis à des fins académiques, dans un objectif de transparence et de reproductibilité scientifique.
